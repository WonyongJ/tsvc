# PDF 병합 도구

여러 PDF 파일을 업로드하고 원하는 페이지만 선택하여 하나의 PDF로 병합할 수 있는 웹 애플리케이션입니다.

## 기능

- 📄 여러 PDF 파일 동시 업로드
- 🖼️ PDF 페이지 썸네일 미리보기
- ✅ 개별 페이지 선택/해제
- 🔄 전체 선택/해제 기능
- 🔀 선택한 페이지 병합
- 📥 병합된 PDF 자동 다운로드
- 📊 병합 진행률 표시

## 사용 방법

1. PDF 파일을 드래그 앤 드롭하거나 클릭하여 업로드
2. 각 PDF의 페이지 썸네일을 확인
3. 병합하고 싶은 페이지를 클릭하여 선택
4. "선택한 페이지 병합" 버튼 클릭
5. 병합된 PDF가 자동으로 다운로드됩니다

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
- PDF.js (PDF 렌더링)
- pdf-lib (PDF 병합)
